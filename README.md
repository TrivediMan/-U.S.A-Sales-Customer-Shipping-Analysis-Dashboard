# 🛒 U.S.A Sales, Customer & Shipping Analysis Dashboard

## 📊 Project Overview
This Power BI project provides a complete analysis of **Sales, Customers, Regions, and Shipping Performance** across the U.S.A.  

The dashboard is divided into **three main sections**:
- Sales Analysis
- Customer & Region Analysis
- Shipping Analysis

It helps businesses track revenue, profit, customer distribution, and shipping efficiency.

---

## 🚀 Key Features

### 🔹 1. Sales Analysis
- Total Revenue: **84.25M**
- Total Profit: **21.68M**
- Total Cost: **62.57M**
- Average Order Value (AOV): **1,685.08**

📈 Insights:
- Year-wise revenue trend (2023–2025)
- Category-wise revenue distribution:
  - Electronics
  - Toys
  - Clothing
  - Home & Garden
  - Books

---

### 🔹 2. Customer & Region Analysis
- Total Customers: **10K**
- Top Performing State: **Colorado**

📊 Features:
- Customer details table (Revenue, Quantity, Gender)
- Interactive map showing state-wise distribution
- Top customers based on revenue

---

### 🔹 3. Shipping Analysis
- Total Orders: **50K**
- Total Shipping Cost: **549.79K**

📦 Insights:
- Orders by shipping status:
  - In Transit
  - Delivered
  - Delayed
  - Returned
- Revenue by shipping method:
  - Overnight
  - Two-Day
  - Express
  - Standard

---

## 🧩 Filters & Interactivity
- Month Filter
- State Filter
- Shipping Status Filter

All visuals are interconnected for dynamic analysis.

---

## 🛠️ Tools & Technologies
- Power BI
- DAX (Data Analysis Expressions)
- Data Modeling
- Interactive Visualizations
